# java-cli-gradle-cucumber-junit-test-hello-world

## Description
A POC for gradle app using JUnit.
Uses cucumber plugin that publishes
report online. JUnit dumps
reports to `reports`.

## Tech stack
- java
- gradle
  - junit
  - cucumber

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
